# what is file handeling in php ?

  1. file handeling is used to handel the file open file | read file and closed file there we used file handeling 

  2. file handeling manage external file to open | write | read and delete file 

  3. file handeling used some function and mode to manage file 


# file handeling function

  1. fopen()
  2. fclose()
  3. fgets()
  4. fread()
  5. fwrite()
  6. unlink()

# file handeling mode that can read | write | append data on file 

  1. r : read only if file does not exist can not create new file
  2. r+ :read | write only if file does not exist can not create new file
  3. w : write only if file does not exist  create new file also
  4. w+: read| write only if file does not exist  create new file also
  5. a : append add new data only if file does not exist  create new file also
  6. a+ : read | write and append data only if file does not exist  create new file also
